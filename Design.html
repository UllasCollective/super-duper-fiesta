Abstract of project

The interface I have built for this Midterm is a personal music player. On the interface page you will find a couple songs I have produced and used as examples.


Design Goals

My goals for this application is to build an interface which can be used to upload and store music for producers. I hope to make this a website that can hold different user accounts, and be able to remember their songs. I have some ideas about the javascript ie. local storage as a function to hold data in the url. (But that'll be version.6)


HTML Tags

The < br > tag was used to make space in between different items on the page.

The < audio > tag is being used to to give control to the song ie. play, pause, and volume.

The < img src= > tag is used to place images into the website. With css, you can resize and place the picture where you want.
